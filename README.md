# 🎮 GameDome

**GameDome** is a multi-game web platform where users can play various classic games, view leaderboards, and manage their profiles. The platform provides a smooth dashboard experience with engaging visuals and intuitive navigation.

## 🚀 Features

- User authentication and login  
- Central dashboard to launch games  
- Single classic games built-in  
- Real-time leaderboard tracking  
- Profile page to manage user information  

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: PHP  
- **Database**: MariaDB  
- **Game Development**: JavaScript Canvas, DOM-based games
